# XGBOOST-AFT ageing clock trained in the UK Biobank

# Introduction
Here we provided three versions of XGBOOST-AFT ageing clock trained in the UK Biobank.

1. Circulating blood clock of 25 blood biomarkers
2. Basic 10 physical measurements clock
3. Enhanced 24 physical measurements Clock

# How to use
1. The demo dataset contains some small datasets generated for demonstration purposes.
2. Predictive function is provided; the required columns are Sex, age0 (age when the measurements were collected) and biomarker features, which must be named the same as demos.
3. Please follow the instructions in 'To_get_AFT_BAAs.R'.
